# DS-zpfp-family-guide — Инструкции для Claude Code

> **Тип репозитория:** `DS/surface`
> **Upstream Pack:** `PACK-kids-learning` — source-of-truth для доменных знаний (8 ZP-принципов, эпизоды, Guard Policy, FM, роли, характеристики мастерства)
> **Upstream Content:** ZP (нулевые принципы), FPF (первые принципы), руководства МИМ (Интеллект-стек, Инженерия мастерства, Системное мышление)

---

## 1. Что это

Прикладное **руководство для взрослого** (родителя, наставника), который хочет выращивать у ребёнка 3-10 лет мышление из нулевых и первых принципов. Руководство = адаптер между программами МИМ и семейной практикой.

**Ключевое различие:**

| PACK-kids-learning | DS-zpfp-family-guide |
|--------------------|---------------------|
| Доменные знания: принципы, методы, роли, FM | Текст руководства для наставника |
| Абстрактная методика (шаблоны эпизодов) | Конкретные тексты, карточки, задания, промпты для бота |
| Source-of-truth | Downstream: берёт из Pack и адаптирует |

## 3. Ключевые правила

1. **Доменные знания берутся из PACK-kids-learning.** Не дублировать, не создавать здесь.
2. **Материалы из руководств МИМ** не копируются — адаптируются под задачу «обучение детей в семье» со ссылками на первоисточники.
3. **Три возрастные зоны:** Младшие (3-5), Старшие дошкольники (5-7), Школьники (7-10). Границы нежёсткие — корректируются по наблюдениям.
4. **Два контура:** взрослый учится сам + взрослый ведёт ребёнка. Оба обязательны.
5. **Бот (DS-instrument)** — отдельный репо. Здесь только контент, промпты и структура.
