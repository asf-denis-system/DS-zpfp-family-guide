# Промпты для бота

Системные промпты для Telegram-бота, помогающего наставнику вести программу развития мышления ребёнка через нулевые принципы (ZP).

## Файлы

| Промпт | Когда | Файл |
|--------|-------|------|
| Скрипт эпизода | Наставник готовится к эпизоду | [episode-script.md](episode-script.md) |
| Анализ лога | Конец недели, обзор | [log-analysis.md](log-analysis.md) |
| Стратегия | Онбординг, перестратегирование | [strategy-generation.md](strategy-generation.md) |

## Принципы

- Бот обращается к взрослому, не к ребёнку
- Тон: уважительный, инженерный, без морализаторства
- Короткие блоки, конкретные действия
- Всегда предлагает записать лог
- Источник доменных знаний: PACK-kids-learning (8 ZP-принципов, эпизоды, Guard Policy, FM, роли, характеристики мастерства)

## Переменные окружения

Все промпты используют общий набор переменных, которые заполняются из профиля семьи при онбординге и обновляются по ходу программы:

| Переменная | Описание |
|---|---|
| `{family_context}` | Профиль семьи: имя ребёнка, возраст, наставники, расписание |
| `{child_name}` | Имя ребёнка |
| `{age}` | Возраст ребёнка (число) |
| `{current_principle}` | Текущий Z-принцип (Z0-Z7) |
| `{mastery_level}` | Уровень по Mastery Map (0-3) |
| `{recent_episodes_summary}` | Краткое содержание последних эпизодов из лога |
| `{track}` | Текущий трек: A (с опорой) или B (самостоятельный) |

## Версионирование

При обновлении промптов: менять версию в заголовке промпта, фиксировать дату, описывать причину изменения в коммите.
