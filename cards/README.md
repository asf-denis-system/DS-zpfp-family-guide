# Карточки эпизодов

Готовые рецепты микро-эпизодов для наставника. Каждая карточка -- один эпизод, готовый к проведению.

## Как пользоваться

1. Выберите принцип, который сейчас в фокусе
2. Выберите возрастную зону ребёнка
3. Выберите трек: A (с опорой) или B (самостоятельный)
4. Подготовьте материалы
5. Проведите эпизод, запишите лог

## Три возрастные зоны

| Зона | Возраст | Длительность | Форма ответа |
|------|---------|-------------|-------------|
| Младшие | 3-5 | 3-7 мин | Покажи, выбери из двух |
| Старшие дошкольники | 5-7 | 7-12 мин | Скажи почему, назови правило |
| Школьники | 7-10 | 10-20 мин | Две версии, как проверить, запиши |

## Два трека

- **Трек A (с опорой):** Взрослый даёт рамку, варианты, сокращает сложность. Для начала, усталости, нового принципа.
- **Трек B (самостоятельный):** Ребёнок объясняет, предлагает версии, переносит. Для бодрости, знакомого принципа.
